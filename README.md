# exercises_c
Collection of solved exercises in C language

## Solved Exercises
- [Geometric Area Calculation](./Area.c)
- A program that calculates the area of ​​a square, a circle, and an equilateral triangle.
- [Simple Calculator](./Calculator.c)
- A program that performs basic arithmetic operations: addition, subtraction, multiplication, and division.
- [Coleman-Liau Readability Index](./Leggibilità.c)
- A program that analyzes user-entered text and calculates the grade level required to understand it. It counts letters, words, and sentences, and applies the Coleman-Liau Index formula to estimate the difficulty of the text.
- [Arithmetic Mean](./media.c)
- A program that calculates the arithmetic mean of a series of numbers entered by the user.
- [Returning Coin Change](./Cash.c)
- A program that calculates the minimum number of coins (quarters, dimes, nickels, pennies) to be returned for a given amount in cents. It uses modular functions for each coin type and includes input validation.
- [Elimination Election Simulation](./runoff.c)
- A program that simulates a multiple-preference elimination voting system. Votes are reassigned until a winner with an absolute majority is found.
